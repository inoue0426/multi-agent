# Multi-Agent

This is the code example to make a multi-agent using AutoGen and Ollama.

## Requirement

```sh
pyautogen                 0.2.28
openai                    1.30.1
```

Also require Ollama. With these, you can use any models you want to use if it is in Ollama.
